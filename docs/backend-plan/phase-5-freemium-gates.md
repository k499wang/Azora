# Phase 5: Freemium Gates

## Goal

Define and enforce what is free versus paid without making the free app feel empty.

## Free

- Account login
- Cloud sync
- Daily breath-hold test
- Daily streak
- Basic breath-hold result
- Basic heart-rate capture
- Basic breathing exercises:
  - Box Breathing
  - 4-7-8 Breathing
  - Resonance
  - Relaxing Breath
- Last 7 days of breath-hold history
- Personal best hold time
- Basic profile
- Shareable result screen
- Safety guidance
- Restore purchases
- Account deletion
- Privacy controls

## Pro

- Unlimited breath-hold history
- Detailed heart-rate graphs and samples
- Full trend analytics:
  - 30 day
  - 90 day
  - all time
- Advanced session insights:
  - lowest BPM during hold
  - recovery curve
  - average improvement
  - consistency score
- Premium breathing techniques
- Custom breathing patterns
- Custom rounds
- Personalized daily plan
- Advanced goal setting
- Streak freezes later
- Premium achievements later
- XP levels later
- Premium themes or visual modes later
- Data export
- More detailed lung age and health score breakdowns

## Must Stay Free

- Login
- Cloud sync
- Safety guidance
- Basic tracking
- Basic streak
- Account deletion
- Privacy/export/delete-account controls
- Restore purchases
- Critical disclaimers

## Gating Behavior

- Locked Pro surfaces show inline upsells.
- Inline upsells route back to the same paywall.
- Do not gate core trust/safety/account features.

## Open Gate Decisions

- Whether custom breathing patterns are Pro from day one.
- Whether Wim Hof is free or Pro.
- Whether Pro leans first into analytics, content, or coaching.

## Exit Criteria

- Free tier feels useful on its own.
- Pro tier has clear upgrade value.
- Gated features consistently route to the same paywall experience.
